# Credit Card Fraud

## Objetivo

O objetivo deste projeto é realizar uma série de análises sobre o conjunto de dados de fraudes de cartão de crédito, utilizando vários algoritmos várias abordagens com aprendizagem supervisionada e não-supervisionada.

## Conjunto de Dados

O conjunto de dados de transações de cartões de crédito fraudulentas pode ser encontrado no [Kaggle](https://www.kaggle.com/mlg-ulb/creditcardfraud/data).
